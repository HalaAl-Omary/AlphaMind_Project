# AlphaMind: Agentic RAG with MCP & Ollama
### Local Agentic AI Financial Analysis Platform
---

## üß† Project Description
**AlphaMind Pro** is an Agentic AI application designed to perform intelligent financial analysis using a locally hosted Large Language Model (LLM).
The system combines **Retrieval-Augmented Generation (RAG)** with **CrewAI agents** and exposes its capabilities through an **MCP Server**, providing structured and context-aware financial insights.

The application allows users to analyze stock market data and receive professional summaries and recommendations through a Streamlit-based user interface.

---

## üéØ Goals and Objectives
- Build an **Agentic AI system** using CrewAI
- Implement a **RAG pipeline** for contextual reasoning
- Run a **local LLM** using Ollama (no external APIs)
- Expose agent capabilities via an **MCP Server**
- Provide an interactive **frontend using Streamlit**

## üè¢ Business Benefits
- Reduced operational costs by avoiding paid LLM APIs
- Faster and more consistent financial insights
- Deployable in private or on-premise environments
- Can be extended for portfolio analysis, risk assessment, or decision support

This is a Financial Analysis system using:
- **CrewAI** for Multi-Agent orchestration.
- **Ollama (Llama 3)** for local LLM processing.
- **ChromaDB** for RAG (Retrieval Augmented Generation).
- **FastMCP** to expose tools via Model Context Protocol.

## Setup
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Run Streamlit: `streamlit run app.py`.
